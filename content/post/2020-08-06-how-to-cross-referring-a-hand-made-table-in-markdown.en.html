To to cross-referring a hand-made table like this:

  $\boldsymbol m_{ij}$   $Pr(z_{ij}=0)$   $Pr(z_{ij}=1)$   $Pr(z_{ij}=2)$     
 ---------------------- ---------------- ---------------- ---------------- 
   (2, 0, 0)              1                  0                  0             
   (1, 1, 0)             0.5                0.5                 0             
   (1, 0, 1)              0                  1                  0             
   (0, 2, 0)              0.25               0.5                0.25           
   (0, 1, 1)               0                 0.5                0.5            
   (0, 0, 2)              0                  0                  1  

Table: (\#tab:label) table name

You chould add (\#tab:label)before the table name, and use \@ref(tab:label)to refer the table 1.

Table 1: table name
\(\boldsymbol m_{ij}\) \(Pr(z_{ij} = 0)\) \(Pr(z_{ij} = 1)\) \(Pr(z_{ij} = 2)\)
(2, 0, 0) 1 0 0
(1, 1, 0) 0.5 0.5 0
(1, 0, 1) 0 1 0
(0, 2, 0) 0.25 0.5 0.25
(0, 1, 1) 0 0.5 0.5
(0, 0, 2) 0 0 1
